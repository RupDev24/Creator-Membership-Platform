## 📄 About the Project
StellarMembr is a decentralized creator monetization platform built on the Stellar Network using Soroban Smart Contracts. The platform enables creators to launch subscription-based membership programs while allowing fans to securely subscribe using XLM through self-custodial Stellar wallets.

The project combines blockchain transparency, decentralized payments, and a modern web experience to provide a scalable alternative to traditional creator membership platforms.

### Core Capabilities
* On-chain creator registration
* Custom membership tier creation
* XLM-based subscriptions
* Wallet-native authentication
* Real-time contract event monitoring
* Decentralized transaction verification
* Subscription lifecycle management

## 🏆 Why StellarMembr?
Traditional creator platforms typically rely on centralized payment processors, recurring platform fees, and limited ownership of subscriber relationships. StellarMembr addresses these challenges by leveraging decentralized smart contracts and direct wallet interactions.

Benefits include:
* Trustless subscriptions
* Transparent on-chain records
* Low transaction costs via Stellar
* Self-custodial wallet ownership
* Global accessibility
* Creator-controlled membership tiers
